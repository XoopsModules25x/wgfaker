# wgFaker
A XOOPS module for generation of faked test data and save them as yaml file.

## Require:
- XOOPS 2.5.11;
- XOOPS Admin 1.2;
- PHP 7.4 or higher;
- MySQL 5.5; -> Minimum Recommended Using InnoDB Version 5.1 or Later
